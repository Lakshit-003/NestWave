# NestWave - Smart Home Solutions

NestWave is a modern web application showcasing smart home solutions and automation services. Built with React and Vite, it features a responsive design and interactive components.

## Features

- 🏠 Modern, responsive landing page
- 🎨 Beautiful UI with Tailwind CSS
- ⚡ Fast development with Vite
- 📱 Mobile-first design
- 🔍 SEO optimized
- 📝 Interactive contact form
- 🖼️ Image gallery
- 💬 Customer testimonials
- ⚙️ Smart home features showcase

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- ESLint
- PostCSS
- Lucide React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone [your-repository-url]
cd nestwave
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and visit:

```
http://localhost:5173
```

### Building for Production

To create a production build:

```bash
npm run build
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
